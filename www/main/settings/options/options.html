Feature colors

If enabled, the map will show a color bar to change the feature color.
If enabled, the map will show a geolocation button.
If enabled, the map will show a "group" button. Activate this button to relate the next feature to the previous feature.
If enabled, the map will show a button to cache the current map extent for offline use.
If enabled, imported features can be deleted with the undo/clear buttons.
If enabled, a window will show up where you can enter additional information after placing a feature on the map.
If enabled, imported features will be exported with newly drawn features, although in a different file.
If enabled, feature colors will be chosen automatically based on group membership.
If enabled, lines and points cannot be grouped together.
Color palette to style map objects. Changes will also affect automatically assigned colors.
